# Aplicação Android/IOS Base

### Requisitos

[Nodejs 22](https://nodejs.org/en)\
[Ionic](https://ionicframework.com/)

---

## Instalação

1. Instalar dependências

```sh
    npm install
    npm install -g typescript
```

2. Configurar ambiente

```sh
    VITE_REST_API=http://192.168.1.250:13090/
```

3. Start

```sh
    npm start
```
